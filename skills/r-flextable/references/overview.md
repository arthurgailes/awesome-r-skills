# Using flextable

This is the primary CRAN vignette for the flextable package.

## Documentation Resources

The flextable package provides two main documentation resources:

1. **User Manual**: Comprehensive guide for working with flextable
   - URL: https://ardata-fr.github.io/flextable-book/
   - Contains: Complete tutorials, examples, and best practices
   - Covers: All aspects from basic tables to advanced formatting

2. **Function Reference**: Complete API documentation
   - URL: https://davidgohel.github.io/flextable/reference/index.html
   - Contains: Help documentation for all 240+ functions
   - Organized by category for easy navigation

## Note

The CRAN vignette serves primarily as a navigation page directing users to the comprehensive external documentation maintained by the package developers. For detailed guidance on using flextable, refer to the resources above or see the other reference files in this skill's references/ directory.
